# 💡 Tech Quiz - Aplicação Interativa

Este projeto é uma aplicação web de Quiz desenvolvida para testar conhecimentos em tecnologia. O foco principal foi o estudo de **Lógica de Programação** e **Manipulação do DOM** com JavaScript puro.

### 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização moderna e layout responsivo.
* **JavaScript (Vanilla)**: Lógica do jogo.

### 🧠 O que eu aprendi neste projeto:
1. **Controle de Fluxo**: Gerenciamento de listas de objetos (perguntas) e iteração de arrays.
2. **Event Listeners**: Captura de interações do usuário para disparar ações em tempo real.
3. **UX Simples**: Criação de feedbacks visuais imediatos para acertos e erros, melhorando a experiência do usuário.
